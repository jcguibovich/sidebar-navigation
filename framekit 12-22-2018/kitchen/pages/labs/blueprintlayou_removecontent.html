For Designers

Sketch Design Assets

The Blueprint set of Sketch library files contains visual language elements and pre-built design elements and components defined as reusable symbols. Enabling Designers to rapidly create UI designs using Blueprint.

Delayed
Due to outdated version of Sketch distributed at Fannie Mae and the breaking changes it brings to that functionality, the Sketch Libraries are currently delayed until the newer version can be approved and packaged for use at Fannie Mae. We are currently working on more basic Sketch assets so designers have a few elements they can use and maintain manually

Quick Start

  • Download the Libraries Coming Soon
  • Launch Sketch (Version 48.2 is Required)
  • In Sketch, Go to Sketch -> Prefences -> Libraries
  • Click Add Library
  • Select, Blueprint Foundation, Blueprint Web, Blueprint Icons and Blueprint Redline.

Understanding the Libraries

Blueprint Foundation :
Holds core DNA Elements of the Design System and thus the DNA of all Symbols within the Sketch Libraries.

Blueprint Web :
This holds components and styles for the Web toolkit. This is the primary library for use in CXD Product Design.

Blueprint Icons :
Blueprint has a growing Utility icon library, called ucons. See Icon documentation for guidelines for their use, but given their update schedule, they are maintained in a separate library.

Blueprint Redline :
This library holds symbols and elements that aid designers in using the design system. For example, guides and UI mockup helpers.

Using the Libraries
When the Blueprint-Web library has been successfully added to Sketch you will have access to the components by going up
Insert > Blueprint-Web > Component
